Ruby Hashes vs. JavaScript Objects

2014. 10. 30

I've discusses about Ruby Hashes few weeks ago here. This week, I've worked with JavaScript for the first time, and found something interesting. JavaScript Objects are very similar to hashes in Ruby.

Hashes in Ruby

JavaScript Objects

JavaScript Objects are data structures that consist of properties. The properties are like a key-value pair in Ruby hashes, but we say 'name' and 'value' for the objects. Unlike Ruby that has data encapsulation and scopes that restricts access to its objects, there is no restriction on accessing the object properties.
