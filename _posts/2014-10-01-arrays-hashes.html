Array vs. Hash

2014. 10. 01

Array

Array can be considered as a linear box that contains multiple elements. It can hold any types of data: numbers, strings, arrays, hashes and more.
Basically array is defined as below and each element has index number.
Index numbers start from 0 and it can be counted reversely from -1 which has the last element. This index number is useful to access each data in an array.

To access an element with its index number, we can write as array[index]. To get number 3 in the array defined in the example, write array[2].

Examples

  • String array: strArray = ['Monday', 'Tuesday', 'Wednesday']
  • Multidimensional array: multiArray = [[1,2,3],[4,5,6],[7,8,9]]
  • strArray << 'Thursday' will add a value to strArray.
    strArray = ['Monday', 'Tuesday', 'Wednesday', 'Thursday']
  • array[4]=7 will change the value in the index number 4.
    array = [1,2,3,4,7]


Hash

Hash is like a dictionary which has pairs of a keys and its value. It also can hold any types of data. It is good to put keys as a symbol. Keys act like index numbers to get its value, but it can also be accessed with index numbers starting from 0 like arrays.

Examples

  • Get a value using a key: hash[:name] is Hailey
  • Change a value of a key: hash[:color] = yellow
    Then, hash = {:name => Hailey, :color => yellow, :city => Vancouver}
  • Add a pair: hash[:school] = 'UBC'
