Ruby Class

2014. 10. 19

Class definition

Class is defined as the below example. It usually has initialize method to initialize the class with/ without arguments.

Variables with a @ sign are classified as instance variables. These variables are accessable within this class. It can be modified, updated. If we want to read or modify its value outside of the class definition, we can set them with attr_reader to read, attr_writer to write different value to it and attr_accessor to do both. Instead, we want to limit the access as we want for protect the values in some case.

Examples

Here are some example use of Pastry class.

apple_pie = Pastry.new(["flour", "butter", "sugar", "apple"], 5) is making an instance of Pastry class. Since we set price as attr_reader attr_reader :price, we can read the value of price outside of the class definition.
